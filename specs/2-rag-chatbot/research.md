# Research for Integrated RAG Chatbot

## Resolved Clarifications

### Testing Framework for Python Backend

-   **Decision**: pytest
-   **Rationale**: pytest is the de facto standard testing framework for Python applications, offering a rich ecosystem of plugins, easy test discovery, and powerful fixtures. It is well-suited for testing FastAPI applications.
-   **Alternatives considered**: unittest (standard library, but pytest is more ergonomic and powerful for modern Python development).
